While you’re in a gdb session, how do you set the arguments that will be passed to the program when it’s run?
当您处于 gdb 会话中时，如何设置在程序运行时传递给程序的参数？
set args 

How do you create a breakpoint?
如何创建断点？
break 

How do you execute the next line of C code in the program after stopping at a breakpoint?
在断点处停止后如何执行程序中的下一行 C 代码？
next

If the next line of code is a function call, you’ll execute the whole function call at once if you use your answer to #3. (If not, consider a different command for #3!) How do you tell GDB that you want to debug the code inside the function (i.e. step into the function) instead?
如果下一行代码是函数调用，则如果您使用 #3 的答案，您将立即执行整个函数调用。 （如果没有，请考虑为 #3 使用不同的命令！）如何告诉 GDB 您想要调试函数内的代码（即 单步进入函数）？
(If you changed your answer to #3, then that answer is most likely now applicable here.)
（如果您将答案更改为#3，那么该答案现在很可能适用于此。）
step

How do you continue the program after stopping at a breakpoint?
程序在断点处停止后如何继续执行？
continue

How can you print the value of a variable (or even an expression like 1+2) in gdb?
如何在 gdb 中打印变量（甚至像 1+2 这样的表达式）的值？
print

How do you configure gdb so it displays the value of a variable after every step?
如何配置 gdb，使其在每一步后显示变量的值？
display

How do you show a list of all variables and their values in the current function?
如何显示当前函数中所有变量及其值的列表？
info locals

How do you quit out of gdb?
如何退出gdb？
quit
